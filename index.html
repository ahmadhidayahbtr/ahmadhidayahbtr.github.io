Tentang

Saya adalah mahasiswa di Universitas Madura fakultas Ekonomi prodi Akutansi. Saya juga seorang web developer di bagian frontend developer. Saya adalah programmer pemula yang sudah menguasai setidaknya 3 bahasa pemrogramman. Bahasa pemrogramman yang saya kuasai HTML, CSS dan Javascript serta framework boostrap dan jquery.

Cerita saya ketika menjadi programmer pemula

Waktu itu masuk kuliah masih kurang 1 bulan. Ketika itu saya merasa tidak punya kegiatan sama sekali sehingga saya berinisiatif untuk mencoba hal baru dari youtube. Awalnya saya tertarik di dunia haacker tetapi saya lihat ada pekerjaan yang tidak kalah hebat dari hacker. Pekerjaan tersebut diantaranya ada web developer,data engeener,UI/UX desaign, dan SEO IT. Saya cari tahu satu persatu sehingga saya lebih tertarik di web developer bagian frontend developer. Pertama yang saya lakukan, saya mencari tahu skill apa yang dibutuhkan oleh seorang frontend developer. Setelah itu, saya belajar bahasa pemrogramman seperti HTML,CSS DAN Javascript. 1 bulan saya belajar html dan 2 bulan saya belajar css. Disitulah saya mulai membuat web sederhana saya. Karena html dan css menurut saya kurang responsif terhadap layar user pengguna maka saya belajar framework css untuk mempermudah saya dalam mengerjakan kodingan saya yaitu bootstrap. Setelah saya belajar bootstarp saya belajar javascript selama 2 bulan dan alhamdulillah saya bisa menggunakan 3 bahasa pemrogramman dan 1 framework css ke dalam web projek saya.
